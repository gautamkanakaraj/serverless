⚙️ Database Connection Settings

Connect your personal Neon PostgreSQL database. We will automatically bootstrap the necessary schemas and tables.

💡 How to create & get your connection string:
  1. Sign up for a free database at neon.tech.
  2. Create a new project (e.g. mini-lambda-db).
  3. On the Neon dashboard homepage, locate the Connection Details widget.
  4. Copy the connection string (starts with postgresql:// or postgres://).
  5. Paste it in the field below and click Save & Connect.
Example: postgresql://gautam:your_password@ep-cool-butterfly.ap-southeast-1.aws.neon.tech/neondb?sslmode=require
Don't have a connection string? Let us provision one for you.
